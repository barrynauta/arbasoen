# Arbasoen Genealogy PDF Examples

This repository contains three example PDFs generated from GEDCOM files, demonstrating genealogy visualization.

## Examples
The following examples are available:
- `[kennedy_nl.pdf](kennedy_nl.pdf)` - Kennedy family tree in Dutch.
- `[kennedy_en.pdf](kennedy_en.pdf)` - Kennedy family tree in English.
- `[royasl92_en.pdf](royasl92_en.pdf)` - Royal 92 family tree in English.

## GEDCOM Sources
The GEDCOM files used for these PDFs are available from:
- [FindMyPast GEDCOM Samples](https://github.com/findmypast/gedcom-samples)
- [Arbre App Public GEDCOMs](https://github.com/arbre-app/public-gedcoms)
